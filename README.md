# 🌸 Perfume Finder

> *Find your signature scent — personalized, filtered, and saved just for you.*

---

## ✨ What is this?

A multi-user Flask web app that helps you discover your perfect fragrance through a personalized quiz. Save your favorites, leave personal notes, and track your scent profile — all in one place.

---

## 🚀 Features

| | |
|---|---|
| 🔍 **Smart Quiz** | Filter by gender, budget, season & mood |
| 💾 **Favorites** | Save perfumes with personal notes |
| 📊 **Dashboard** | Your top scent note & daily pick |
| 🔐 **Auth** | Register, login, logout with sessions |
| 💡 **Alternatives** | Budget picks & luxury inspirations |

---

## 🛠️ Tech Stack

```
Backend   →  Python, Flask
Database  →  SQLite (raw SQL, no ORM)
Frontend  →  HTML, CSS, JS (Jinja2)
Auth      →  Session-based + password hashing
Tests     →  Python unittest
```

---

## ⚙️ Setup

```bash
git clone https://github.com/dilaycelikk/perfume-finder.git
cd perfume-finder
pip install flask
python app.py
```

---

## 🧪 Tests

```bash
python -m unittest tests/test_logic.py -v
```
